如何遍历Object

不要使用 for in，参考这里

for (const [key, val] of Object.entries(obj)) {
    console.log(key, val)
}

当然，const是常量，如果你还需要在循环中对他进行改变，可以使用var变量

这个Object.entries(obj))是关键，它可以把obj（一个不可迭代的对象）转成一个可以迭代的对象
